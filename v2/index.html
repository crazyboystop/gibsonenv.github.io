Gibson Env V2

Gibson Env V2 is a significantly improved version over Gibson Env. Gibson Env V2 has significantly faster rendering speed, which makes large-scale training possible. It also allows for more complicated interactions between the agent and the environment, such as picking up and placing objects, or opening doors and cabinets. This environment opens up new venues for jointly training base and arm policies, allowing researchers to explore the synergy between manipulation and navigation.
